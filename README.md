# Projekt-Aplikacji-Mobilnej

Głównym założeniem projektu było opracowanie interaktywnej gry zręcznościowej, w której użytkownik steruje postacią wykonującą skoki, unikając przeszkód oraz zdobywając punkty. Inspirację dla projektu stanowiły popularne gry mobilne, takie jak „Pou” oraz „Ice Cream Jump”.


Celem realizacji aplikacji było przede wszystkim zdobycie praktycznej wiedzy z zakresu tworzenia gier mobilnych oraz sprawdzenie czy platforma Android stanowi odpowiednie środowisko do projektowania i implementacji tego typu rozwiązań.
